Library Management System (LMS)

This is a simple console-based Library Management System written in C++.  
I made this project to practice basic Data Structures and understand how vectors and structures work in C++.

-> Features

Add a new book to the library  
Show all books in the library  
Simple menu-driven program  

-> Concepts used

C++ structures  
STL vector (dynamic array)  
Insertion of data  
Traversal of vector  

-> How to run the program

Compile the program

g++ -std=c++11 main.cpp

Run the program

./a.out

-> Example menu

1 Add Book  
2 Show Books  
3 Exit  

Made while learning C++ and basic DSA.
